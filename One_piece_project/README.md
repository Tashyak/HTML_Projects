# 🏴‍☠️ One Piece Landing Page

A responsive **One Piece-themed landing page** built using **HTML5** and **CSS3**. This project is inspired by the world of *One Piece* and showcases modern web design techniques with clean layouts, smooth styling, and responsive design.

## 🌟 Live Demo

🔗 https://onepieceproject.vercel.app/#home

---

## 📖 About the Project

This project is a fan-made landing page created to practice front-end web development using only HTML and CSS.

The website features a visually appealing design inspired by the One Piece universe while focusing on responsive layouts, clean code structure, and an engaging user interface.

> **Note:** This project is created for educational and portfolio purposes only. All One Piece characters, logos, and related artwork belong to their respective copyright owners.
